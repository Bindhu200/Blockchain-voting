# Blockchain Voting System (Healthcare-Inspired Secure Platform)

A secure online voting platform utilizing simulated blockchain technology for transparency and tamper-proof records. Features user authentication, one-vote-per-user, real-time results, and modern UI — similar to secure healthcare management systems.

## Features
- User registration & login (secure password hashing + JWT)
- One vote per user (tamper detection)
- Simulated blockchain with SHA-256 hashing and chain validation
- Live vote results dashboard
- Professional, responsive design

## Technologies Used
- Python, Flask
- SQLite (user database)
- Cryptography (password hashing)
- PyJWT (authentication)
- HTML/CSS/Jinja2 (frontend)
